💡 直观

store和组件用法一样。精心设计的API使您能够写出结构良好的store.

🔑 类型安全

类型是可推断的，这意味着即使在 JavaScript 中， store也可以为您提供自动补全！

⚙️ 支持开发者工具

Pinia 与 Vue devtools 紧密连接，为您提供更好的 Vue 2 和 Vue 3 开发体验。

🔌 可扩展

对store的更改是响应式的，这使得Pinia更易于通过事务、本地存储进行扩展。

🏗 模块化设计

可构建多个store并使您的打包程序能够进行自动拆分。

📦 极其轻量

Pinia 约 1kb，您甚至会忘记它的存在！!
